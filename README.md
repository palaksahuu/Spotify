#  Spotify Clone – Web Music Player

A simple **Spotify-inspired music player** built using **HTML, CSS, and JavaScript**.  
This project demonstrates how a browser-based music player works, including features like playing songs, pausing, navigating tracks, and displaying song progress.

It is a **frontend project** that replicates basic music player functionality similar to Spotify.

---

##  Features

-  Play and pause songs
-  Next and previous song navigation
-  Song progress bar with seek functionality
-  Dynamic song list display
-  Song cover images
-  Animated "Now Playing" GIF indicator
-  Responsive design using CSS

---

##  Technologies Used

- **HTML5** – Structure of the web application  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla JS)** – Application logic and interactivity  
- **Font Awesome** – Icons for music controls  
- **Google Fonts** – Typography  

---

##  How It Works

The application uses the **HTML Audio API** to control music playback.

### 1️ Play / Pause
- Clicking the play button starts or pauses the current song.
- The icon switches between play and pause.

### 2️ Progress Bar
- Shows the current playback progress.
- Users can move the slider to jump to different parts of the song.

### 3️ Song Selection
- Clicking a song from the list loads and plays that track.

### 4️⃣ Next / Previous Buttons
- Allows navigation between songs.
